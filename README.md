## jakubthedeveloper

Embedded C · C++ · Python · MIDI · analog electronics · whatever it takes.

I don't pick a language. I pick a problem, then reach for the right tool —
even if that tool is a soldering iron.

My projects live somewhere between software and hardware, between "this 
probably works" and "this definitely shouldn't work but does." I go deep.
Not because I have to, but because the surface is boring.

DIY isn't a budget constraint. It's a worldview.
If something doesn't exist yet, that's an invitation, not a warning.
